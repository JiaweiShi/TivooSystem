SSRI~ Workshop: Using LaTeX

Start time: 09/29/11 03:00
End time: 09/29/11 05:00
Description: LaTeX is becoming the most widely used document processing system in the social sciences with good reason: it generates publication quality documents, allows for precise mathematical notation and equations, has a built in citation manager and it is FREE. This workshop will cover how to build a basic document with an abstract, tables, figures and citations. No previous knowledge of LaTeX or programming is needed.
